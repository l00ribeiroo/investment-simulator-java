# Investment Simulator (Java)

Projeto em Java que simula o crescimento de investimentos com aportes mensais e juros compostos.

## Funcionalidades

- cálculo do valor final do investimento
- aplicação de juros compostos
- entrada de dados pelo usuário

## Tecnologias

- Java
- IntelliJ IDEA
- Git e GitHub

## Exemplo

Valor inicial: 1000  
Aporte mensal: 200  
Taxa: 1%  
Tempo: 12 meses  

Resultado: R$ 3663,33

## Objetivo

Projeto desenvolvido para praticar lógica de programação e estruturas de repetição.
